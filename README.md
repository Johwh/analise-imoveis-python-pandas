# Análise de Imóveis com Python e Pandas

Projeto de análise de dados desenvolvido com Python e Pandas utilizando uma base de imóveis para aluguel.

## Objetivo

Realizar limpeza, tratamento e análise exploratória dos dados para identificar padrões e gerar insights relevantes.

## Tecnologias Utilizadas

- Python
- Pandas
- Google Colab
- Git e GitHub

## Etapas do Projeto

- Importação da base de dados CSV
- Tratamento de valores nulos
- Remoção de registros duplicados
- Filtragem de dados inconsistentes
- Análise dos tipos de imóveis e médias de aluguel por bairro

## Estrutura

```bash
data/
├── aluguel.csv
└── aluguel_tratado.csv

src/
└── analise.py
```

## Resultado

Após o processamento, foi gerada uma nova base de dados tratada (`aluguel_tratado.csv`) pronta para análises futuras.

## Aprendizados

Neste projeto, desenvolvi habilidades em manipulação, limpeza e análise de dados com Pandas, além da organização de projetos para portfólio.
